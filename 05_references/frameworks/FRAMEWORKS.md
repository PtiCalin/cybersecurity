# Security Frameworks

Annotated notes on attack and defense frameworks.

## Categories

- **MITRE-ATT&CK/** — Adversary tactics, techniques, and procedures
- **Kill-Chain/** — Cyber Kill Chain and attack lifecycle stages
- **Diamond-Model/** — Intrusion analysis and threat modeling

## Usage

Create framework notes at different granularities:

**Overview notes:**
- `mitre-attack-overview.md`
- `kill-chain-framework.md`

**Detailed notes:**
- `mitre-attack-initial-access.md`
- `mitre-attack-persistence.md`
- `kill-chain-reconnaissance-phase.md`

## Cross-Referencing

Frameworks should connect to:
- Theory (attack/defense concepts)
- Examples (real-world applications)
- CVEs (vulnerability exploitation within framework context)
