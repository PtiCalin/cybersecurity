# Technical Documentation

Annotated notes on protocols, tools, and platform documentation.

## Categories

- **protocols/** — Network protocols (TCP/IP, TLS, HTTP, DNS, etc.)
- **tools/** — Security tool documentation and usage guides
- **platforms/** — Operating systems, cloud platforms, security platforms

## Purpose

These notes extract security-relevant information from technical specs:
- Protocol vulnerabilities and attack surfaces
- Tool capabilities and limitations
- Platform security features and hardening guides
- Configuration best practices

## Integration

Documentation notes feed directly into:
- `01_theory/` for protocol-level security concepts
- `02_examples/` for attack and defense demonstrations
- `04_resources/tools/` for tool catalog entries
