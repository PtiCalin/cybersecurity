# Academic Papers

Annotated notes on cybersecurity research papers.

## Organization

- **offensive/** — Attack techniques, exploitation, offensive research
- **defensive/** — Detection, response, hardening, defensive strategies
- **foundational/** — Core concepts, cryptography, theory, methodology

## Companion to Literature Folder

The `05_references/literature/` folder contains actual PDF files. Create corresponding notes here using the reference template.

**Workflow:**
1. Add PDF to `../literature/`
2. Create note here: `cp ../_reference-note-template.md ./[category]/[paper-name].md`
3. Link between note and PDF in the note's metadata

## Naming Convention

Format: `[first-author-lastname]-[year]-[short-title].md`

Examples:
- `anderson-2001-security-engineering.md`
- `schneier-1996-applied-cryptography.md`
