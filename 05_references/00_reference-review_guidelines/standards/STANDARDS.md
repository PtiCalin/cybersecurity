# Security Standards

Annotated notes on industry security standards and best practices.

## Categories

- **OWASP/** — Web application security standards (Top 10, ASVS, SAMM)
- **NIST/** — NIST Cybersecurity Framework, SP 800 series
- **ISO/** — ISO 27001, 27002, and related standards
- **CIS/** — CIS Controls and benchmarks

## Usage

Create notes using the reference template:
```bash
cp ../\_reference-note-template.md ./[category]/[standard-name].md
```

Example:
```bash
cp ../\_reference-note-template.md ./OWASP/owasp-top-10-2021.md
```
