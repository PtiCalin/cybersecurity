## Learning Path

**Recommended study order:**

1. TCP/IP Fundamentals — Understand the protocol stack
2. Network Protocols — Know common services
3. DoS and DDoS Attacks — Critical availability threat
4. IP Spoofing — Address-based attacks
5. Man-in-the-Middle — Interception techniques
6. Firewall Fundamentals — Primary network defense
7. Intrusion Detection Systems — Active monitoring

**Prerequisites:**

- [CIA Triad](../00-foundations/cia-triad.md) — Understanding availability
- Basic networking concepts

**Leads to:**

- [Offensive Security](../03-offensive-security/) — Attack techniques
- [Defensive Security](../04-defensive-security/) — Detection and response
