## Learning Path

**Cross-domain specialization** — Choose based on career path:

**Cloud Security Path:**
1. Cloud Security Fundamentals
2. Container Security
3. Multi-Cloud Security

**IoT/Embedded Path:**
1. IoT Security
2. Wireless Security
3. SCADA & ICS Security

**Compliance Path:**
1. Privacy Engineering
2. GDPR Compliance
3. Industry-specific regulations (HIPAA, PCI-DSS)

**Prerequisites:** 
- Core security domains (foundations, network, application)
- Domain-specific technical knowledge (e.g., cloud platforms, embedded systems)
