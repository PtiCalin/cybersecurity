## Learning Path

**Recommended study order:**

**Foundations:**
1. MITRE ATT&CK — Understanding adversary tactics (offensive view)
2. Defense in Depth — Layered security architecture
3. NIST Cybersecurity Framework — Comprehensive security program

**Management:**
4. ISO 27001 — Formal ISMS structure
5. Risk frameworks (NIST RMF, FAIR) — Systematic risk management
6. CIS Controls — Prioritized security actions

**Advanced:**
7. Zero Trust Architecture — Modern security model
8. MITRE D3FEND — Defensive techniques catalog
9. Maturity models — Measuring security capability

**Prerequisites:** 
- Core security knowledge across all domains
- Understanding of organizational security needs

**Leads to:** 
- Security management and governance roles
- Compliance and audit positions
- Security architecture design

## Framework Selection Guide

| Organization Type | Primary Frameworks |
|-------------------|-------------------|
| **Enterprise (general)** | NIST CSF, ISO 27001, CIS Controls |
| **Government/Defense** | NIST RMF, CMMC, FedRAMP |
| **Healthcare** | HIPAA Security Rule, HITRUST |
| **Finance** | PCI-DSS, FFIEC, SOX |
| **Technology Startup** | NIST CSF (simplified), CIS Top 5 |
| **Critical Infrastructure** | NIST CSF, ICS/SCADA frameworks |
