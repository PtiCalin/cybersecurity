## Learning Path

**Recommended study order:**

1. Intrusion Detection — Understanding how attacks are detected
2. Security Monitoring — Continuous observation
3. System Hardening — Proactive defense
4. Incident Response — Handling breaches
5. Threat Hunting — Active defense
6. Security Operations — Operational security

**Prerequisites:**

- [Offensive Security](../03-offensive-security/) — Know what you're defending against

**Leads to:**

- Complete defensive capabilities
- SOC analyst readiness
- Incident response proficiency
