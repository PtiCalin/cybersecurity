## Learning Path

**Recommended study order:**
1. CIA Triad — Foundation of all security
2. Security Properties — Extended security characteristics
3. Threat Modeling Basics — Understanding attackers
4. Risk Analysis — Systematic risk assessment
5. Security Governance — Organizational security
6. Defense in Depth — Practical security architecture

**Prerequisites:** Basic computer literacy

**Leads to:** All other security domains build on these foundations
