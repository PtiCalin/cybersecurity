## Template Usage Notes

**When creating a new exercise from this template:**

1. Delete this section
2. Choose exercise type and difficulty
3. Define clear, measurable objectives
4. Document environment setup completely
5. Test the challenge yourself multiple times
6. Create progressive hints (conceptual → technical → specific)
7. Write complete solution with alternative approaches
8. Document common mistakes from testing
9. Add defensive analysis even for offensive exercises
10. Link to related theory, examples, tools, references
11. Create variations for extended practice
12. Have others attempt and provide feedback

**Quality checklist:**

- [ ] Objectives are specific and measurable
- [ ] Environment setup is reproducible
- [ ] Success criteria are unambiguous
- [ ] Progressive hints don't give away solution
- [ ] Solution includes reasoning, not just steps
- [ ] Alternative approaches documented
- [ ] Defensive perspective included
- [ ] Common mistakes from testing noted
- [ ] All links work and are relevant
- [ ] Variations provide extended practice
- [ ] Tested by at least one other person
- [ ] Fair difficulty (no guessing required)
